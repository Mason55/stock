# Stock Analysis System

一个功能强大的股票分析系统，集成技术分析、基本面分析和情绪分析。

## 功能特性

- 多数据源集成 (新浪、东方财富、腾讯、同花顺)
- 20+专业技术指标分析
- 基本面财务数据分析  
- 市场情绪分析
- 智能投资建议
- 风险等级评估

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 运行演示
python comprehensive_demo.py

# 交互式分析
python enhanced_analyzer_demo.py --interactive
```

## 系统架构

- `src/core/` - 核心分析模块
- `src/analyzers/` - 分析器实现
- `data/` - 数据存储
- `tests/` - 测试用例
